Guide de l'organisateur
1 — Créer une course

Cliquez sur Mes courses en haut à droite, puis Créer une course. Donnez-lui un nom. Un code unique est généré automatiquement — c'est ce code que les participants utiliseront pour rejoindre la course.

2 — Configurer la course

Dans l'onglet Config, renseignez le nom, l'unité, la durée du chrono et le nombre de balises prévu. Enregistrez avant de passer à la suite.

3 — Préparer les balises physiques

Confectionnez vos balises manuellement et placez-les sur le terrain à l'endroit de votre choix. Chaque balise doit afficher un numéro et un code visible par le participant (ex : BALISE 1 — KAVKA).

Exemple de balise fabriquée et placée sur le terrain
4 — Saisir les postes dans l'app

Dans l'onglet Postes, ajoutez chaque balise : numéro, nom du lieu, code balise (doit correspondre exactement à celui affiché sur la balise physique) et position GPS. Pour la position GPS, rendez-vous sur place et cliquez sur Ma position, ou placez le marqueur manuellement sur la carte.

Ajoutez ensuite les questions QCM qui seront posées au participant lorsqu'il validera la balise.

La dernière balise est automatiquement définie comme point d'arrivée commun à tous les participants.

5 — Ajouter les joueurs

Dans l'onglet Joueurs, ajoutez le pseudo de chaque participant. Un QR code personnel est généré pour chacun — imprimez-le ou partagez-le. Le participant scanne ce QR code pour rejoindre la course automatiquement avec le bon ordre de rotation.

6 — Lancer la course

Remettez à chaque participant son QR code. Au signal, ils scannent et partent. Suivez leur progression en temps réel dans l'onglet Classement → Suivi en temps réel.

7 — Fin de course

Consultez le classement final dans l'onglet Classement. Vous pouvez arrêter manuellement le chrono d'un participant si nécessaire. Le classement se fait par nombre de balises validées puis par temps ajusté (bonnes réponses = bonus, mauvaises = malus).

Regles du jeu — COPA
Objectif

Chaque participant doit trouver et valider toutes les balises disseminees sur le terrain dans un temps imparti. A chaque balise, il repond a des questions diverses choisies par l'organisateur. Son temps final est ajuste en fonction de ses reponses.

Deroulement

Avant le depart, l'encadrant remet a chaque participant son QR code personnel. Le participant scanne ce QR code pour rejoindre la course automatiquement.

Chaque participant suit un ordre de passage different. La derniere balise est le point final commun a tous.

Sur chaque balise, un code est affiche. Le participant le saisit dans l'application. Il n'a qu'une seule chance de repondre par question.

Les balises

Sur le terrain, des balises sont placees a differents endroits. Chacune affiche un numero et un code a saisir dans l'application pour valider le poste.

Exemple de balise a trouver sur le terrain

Classement

Par nombre de balises validees puis par temps final ajuste. En cas d'egalite, le meilleur temps est classe premier.

Regles a respecter

Le participant progresse seul, sans aide exterieure. Toute communication sur les codes ou les reponses est interdite. L'encadrant peut arreter le chronometre a tout moment.

Navigation

L'application integre une boussole GPS indiquant la direction et la distance vers la prochaine balise. Autorisez la localisation sur votre telephone. Calibrez la boussole en faisant un 8 dans l'air avec le telephone.
