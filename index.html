Regles du jeu — COPA
Objectif

Chaque participant doit trouver et valider toutes les balises disseminees sur le terrain dans un temps imparti. A chaque balise, il repond a des questions tactiques ou techniques. Son temps final est ajuste en fonction de ses reponses.

Deroulement

Avant le depart, l'encadrant remet a chaque participant son QR code personnel. Le participant scanne ce QR code pour se connecter a l'application et demarrer son chronometre.

Chaque participant suit un ordre de passage different sur les balises intermediaires afin d'eviter que les equipes se suivent. La derniere balise est le point de controle final commun a tous.

Sur chaque balise, un code est affiche. Le participant saisit ce code dans l'application. Les questions s'affichent une par une. Il n'a qu'une seule chance de repondre, sans retour possible.

Systeme de bonus et malus
+1 min
par bonne reponse
-1 min
par mauvaise reponse

Temps final = temps brut - bonnes reponses x 60s + mauvaises reponses x 60s.

Classement

Le classement est etabli par nombre de balises validees puis par temps final ajuste. En cas d'egalite, le meilleur temps est classe premier.

Regles a respecter

Le participant progresse seul, sans aide exterieure. Toute communication sur les codes ou les reponses est interdite. L'encadrant peut arreter le chronometre d'un participant a tout moment.

Navigation

L'application integre une boussole GPS indiquant la direction et la distance vers la prochaine balise. Le participant doit autoriser la localisation sur son telephone.
